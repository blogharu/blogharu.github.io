문제점

atom을 저장할 때 가장 마지막의 비어있는 줄들이 삭제되고 한 줄만 남는다.

해결책

  1. atom에서 설정 창에 간다 (ctrl + ',')
  2. packages 버튼을 클릭한다
  3. install packages 아래에 whitespace 입 후 core
     packages 에서 나온 whitespace 의 settings
     버튼을 클릭한다
  4. Ensure single trailing newline 체크 박스를
     해제한다.
