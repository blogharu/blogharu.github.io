yaml이란

yaml은 데이터 직렬화 언어이다. 데이터 직렬화 (data serialization) 이란 복잡한 데이터 구조를 바이트 스트림으로 바꾸는 절차이다. 바이트 스트림으로 바뀐 데이터는 저장, 데이터 전송등에 이용된다.

yaml의 특징

데이터 직렬화 Data Serialization 데이터 직렬화는 복잡한 데이터 구조를 바이트 스트림으로 바꾸는 프로세스다. 바이트 스트림으로 바뀐 데이터는 저장, 전송 등등에 쓰인다. 이해를 돕기위해 예를 들자면 다음 그림과 같이 복잡한 클레스를 다음과 같은 데이터 스트럭처가 있다고 생각해보자.
