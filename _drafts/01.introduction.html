* html, CSS 등등 웹 개발 관련 지식이 있어야 이해가 쉽습니다. *

* jekyll은 공식적으로는 windows를 지원하지 않습니다. jekyll을 사용할 수는 있으나 오류가 발생할 가능성이 있습니다. *

저는 jekyll을 이용하여 jekyll 테마를 사용하지 않고 웹사이트를 만드는 방법을 공부하고자 합니다. 여기에는 제가 공부하면서 알게 된 내용을 정리하도록 하겠습니다.

저는 여러분이 ruby 2.4 버전 이상을 받으셨고, jekyll을 설치하셨고, gem을 통하여 jekyll과 bundler를 설치했다는 가정하에 진행하도록 하겠습니다.

다음 코드를 cmd 또는 terminal에서 문제없이 구동할 수 있으시면 준비가 되신 겁니다. 관리자 권한을 필요로 할 수 있으니 주의하시길 바랍니다.
{% highlight ruby %} gem install jekyll bundler jekyll new myblog cd myblog bundle exec jekyll serve {% endhighlight %}
