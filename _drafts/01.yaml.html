yaml이란?

yaml은 YAML Ain't markup language 의 약어이고 데이터 직렬화 언어이다. yaml은 사람 친화적이고 다른 프로그래밍 언어에서 잘 작동되게 디자인되었다.

yaml의 특징

  1. yaml은 대소문자를 구분한다.
  2. 확장자로 .yaml, .yml을 사용한다.
  3. yaml은 들여쓰기할 때 tab 키를 허용하지 않는다.

yaml

list 
[var1, var2, var3]
== - var1
   - var2
   - var3

yaml 오브젝트

associative arrays 
{key:val, key2:val2}
 == - key: var
    - key2: var2
